# Начало работы

> **Тип:** Guide  
> **Цель:** Быстрый старт с проектом  
> **Аудитория:** Новые разработчики

## Обзор

Это руководство поможет установить и запустить проект «Ассистент» на локальной машине.

## Предварительные требования

- [ ] Node.js 22+ ([скачать](https://nodejs.org/))
- [ ] pnpm 9+ (`npm install -g pnpm`)
- [ ] Git

## Шаги

### Шаг 1: Клонирование репозитория

```bash
git clone <repository-url>
cd assistant
```

### Шаг 2: Установка зависимостей

```bash
pnpm install
```

### Шаг 3: Запуск dev-сервера

```bash
pnpm dev
```

Приложение будет доступно по адресу: http://localhost:5173

### Шаг 4: Проверка

Откройте браузер и перейдите на http://localhost:5173. Вы должны увидеть лендинг приложения.

## Структура проекта

```
assistant/
├── src/              # Исходный код
│   ├── components/   # React компоненты
│   ├── pages/        # Страницы
│   ├── lib/          # Утилиты
│   └── providers/    # React провайдеры
├── docs/             # Документация
├── public/           # Статические файлы
└── reference/        # Референс (Pennora)
```

## Основные команды

| Команда          | Описание                |
| ---------------- | ----------------------- |
| `pnpm dev`       | Запуск dev-сервера      |
| `pnpm build`     | Production сборка       |
| `pnpm preview`   | Превью сборки           |
| `pnpm lint`      | Проверка ESLint         |
| `pnpm format`    | Форматирование Prettier |
| `pnpm typecheck` | Проверка типов          |

## Частые проблемы

### pnpm не установлен

```bash
npm install -g pnpm
```

### Порт 5173 занят

Vite автоматически выберет следующий свободный порт. Или укажите явно:

```bash
pnpm dev --port 3000
```

### Ошибки TypeScript

Убедитесь, что установлены все зависимости:

```bash
pnpm install
```
