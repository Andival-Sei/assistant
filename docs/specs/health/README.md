# Модуль: Health

> **Статус:** Stub  
> **Приоритет:** Medium  
> **Зависимости:** Auth (Supabase)

## Обзор

Модуль здоровья позволяет пользователям отслеживать показатели здоровья и получать рекомендации на основе данных. В текущей версии — заглушка для будущей реализации.

## Цели

- Отслеживание базовых показателей здоровья
- Визуализация трендов
- AI-рекомендации на основе данных (будущее)

## Scope (планируется)

### В scope

- Ввод показателей (вес, давление, пульс, etc.)
- История измерений
- Графики и тренды
- Напоминания об измерениях

### Вне scope

- Интеграция с фитнес-трекерами
- Медицинские рекомендации
- Диагностика

## Планируемые показатели

| Показатель | Единица   | Частота    |
| ---------- | --------- | ---------- |
| Вес        | кг        | Ежедневно  |
| Давление   | мм рт.ст. | По желанию |
| Пульс      | уд/мин    | По желанию |
| Сон        | часы      | Ежедневно  |
| Шаги       | шт.       | Ежедневно  |
| Вода       | мл        | Ежедневно  |

## Интеграция с другими модулями

- **Family** — семья может видеть показатели здоровья друг друга (с разрешения)

## Статус

Модуль находится в стадии планирования. Реализация запланирована на Фазу 3 (см. [roadmap](../../meta/roadmap.md)).
