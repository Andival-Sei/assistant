# Roadmap: Ассистент

**Версия:** 0.1.0  
**Статус:** Начало разработки  
**Цель:** MVP с базовым модулем финансов

---

## Обзор

«Ассистент» — приложение для управления финансами, здоровьем, списками и семейным режимом. Roadmap описывает путь от текущего состояния (лендинг) к полноценному продукту.

---

## Фаза 1: Инфраструктура

> Цель: Подготовить проект к разработке модулей

### 1.1 Supabase Setup

- [ ] Создать Supabase проект
- [ ] Настроить Auth (email/password)
- [ ] Создать базовые таблицы (accounts, transactions, categories)
- [ ] Настроить RLS политики
- [ ] Добавить Supabase клиент в проект

### 1.2 Авторизация

- [ ] Страница входа (`/login`)
- [ ] Страница регистрации (`/register`)
- [ ] Защита роутов (PrivateRoute)
- [ ] Редиректы (логин → дашборд, логаут → лендинг)

### 1.3 TanStack Query

- [ ] Настроить QueryClient
- [ ] Создать QueryProvider
- [ ] Базовая структура query keys

### 1.4 Базовые UI компоненты

- [ ] Перенести компоненты из Pennora (Input, Card, Select, etc.)
- [ ] Адаптировать под текущий проект
