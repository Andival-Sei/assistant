# Finance: User Stories

User stories для модуля финансов.

## Счета (Accounts)

### US-F001: Создание счёта

**Как** пользователь  
**Я хочу** создать новый счёт  
**Чтобы** отслеживать баланс отдельно (карта, наличные, etc.)

**Критерии приёмки:**

- [ ] Можно указать название счёта
- [ ] Можно выбрать тип счёта (карта, наличные, другое)
- [ ] Можно указать начальный баланс
- [ ] Можно выбрать валюту счёта
- [ ] Счёт появляется в списке после создания

### US-F002: Просмотр списка счетов

**Как** пользователь  
**Я хочу** видеть все свои счета  
**Чтобы** понимать общую финансовую картину

**Критерии приёмки:**

- [ ] Отображается список всех счетов
- [ ] Для каждого счёта показан баланс
- [ ] Показан общий баланс (с конвертацией валют)
- [ ] Можно перейти к деталям счёта

### US-F003: Редактирование счёта

**Как** пользователь  
**Я хочу** изменить данные счёта  
**Чтобы** актуализировать информацию

**Критерии приёмки:**

- [ ] Можно изменить название
- [ ] Можно изменить тип счёта
- [ ] Нельзя изменить валюту (есть транзакции)
- [ ] Изменения сохраняются

### US-F004: Удаление счёта

**Как** пользователь  
**Я хочу** удалить счёт  
**Чтобы** убрать неиспользуемые счета

**Критерии приёмки:**

- [ ] Запрос подтверждения перед удалением
- [ ] Нельзя удалить счёт с транзакциями (или предложить удалить их)
- [ ] Счёт исчезает из списка после удаления

## Транзакции (Transactions)

### US-F010: Добавление расхода

**Как** пользователь  
**Я хочу** быстро добавить расход  
**Чтобы** вести учёт трат

**Критерии приёмки:**

- [ ] Можно указать сумму
- [ ] Можно выбрать счёт списания
- [ ] Можно выбрать категорию
- [ ] Можно добавить описание (опционально)
- [ ] Дата по умолчанию — сегодня
- [ ] Баланс счёта уменьшается

### US-F011: Добавление дохода

**Как** пользователь  
**Я хочу** добавить доход  
**Чтобы** отслеживать поступления

**Критерии приёмки:**

- [ ] Можно указать сумму
- [ ] Можно выбрать счёт зачисления
- [ ] Можно выбрать категорию дохода
- [ ] Можно добавить описание
- [ ] Баланс счёта увеличивается

### US-F012: Перевод между счетами

**Как** пользователь  
**Я хочу** перевести деньги между своими счетами  
**Чтобы** отразить реальное перемещение средств

**Критерии приёмки:**

- [ ] Можно выбрать счёт-источник
- [ ] Можно выбрать счёт-получатель
- [ ] Можно указать сумму
- [ ] При разных валютах — показать курс конвертации
- [ ] Баланс источника уменьшается
- [ ] Баланс получателя увеличивается

### US-F013: Просмотр списка транзакций

**Как** пользователь  
**Я хочу** видеть историю транзакций  
**Чтобы** анализировать свои финансы

**Критерии приёмки:**

- [ ] Список отсортирован по дате (новые сверху)
- [ ] Показан тип транзакции (доход/расход/перевод)
- [ ] Показана категория
- [ ] Показана сумма (цветом: зелёный/красный)
- [ ] Можно фильтровать по периоду
- [ ] Можно фильтровать по категории
- [ ] Можно фильтровать по счёту

### US-F014: Редактирование транзакции

**Как** пользователь  
**Я хочу** изменить транзакцию  
**Чтобы** исправить ошибку

**Критерии приёмки:**

- [ ] Можно изменить сумму
- [ ] Можно изменить категорию
- [ ] Можно изменить описание
- [ ] Можно изменить дату
- [ ] Балансы пересчитываются

### US-F015: Удаление транзакции

**Как** пользователь  
**Я хочу** удалить транзакцию  
**Чтобы** убрать ошибочную запись

**Критерии приёмки:**

- [ ] Запрос подтверждения
- [ ] Баланс восстанавливается
- [ ] Транзакция исчезает из списка

- [ ] Видна иерархия (родительские/дочерние)

- [ ] Можно выбрать родительскую категорию
- [ ] Можно выбрать иконку
- [ ] Категория появляется в списке

### US-F022: Редактирование категории

- [ ] Можно изменить иконку
- [ ] Можно изменить родительскую категорию
- [ ] Нельзя изменить тип (есть транзакции)

### US-F023: Удаление категории

**Как** пользователь  
**Я хочу** удалить категорию  
**Чтобы** убрать неиспользуемые

**Критерии приёмки:**

- [ ] Нельзя удалить системные категории
- [ ] Нельзя удалить категорию с транзакциями
- [ ] Запрос подтверждения

## Статистика (Statistics)

### US-F030: Просмотр баланса

**Как** пользователь  
**Я хочу** видеть общий баланс  
**Чтобы** понимать финансовое состояние

**Критерии приёмки:**

- [ ] Показан общий баланс в выбранной валюте
- [ ] Показан баланс по типам счетов (карты, наличные)
- [ ] Автоматическая конвертация валют

### US-F031: Статистика за период

**Как** пользователь  
**Я хочу** видеть статистику за месяц  
**Чтобы** анализировать расходы

**Критерии приёмки:**

- [ ] Показаны общие доходы за период
- [ ] Показаны общие расходы за период
- [ ] Показан чистый результат (доходы - расходы)
- [ ] Можно выбрать период (месяц, год, произвольный)

### US-F032: Расходы по категориям

**Как** пользователь  
**Я хочу** видеть разбивку расходов по категориям  
**Чтобы** понять, на что уходят деньги

**Критерии приёмки:**

- [ ] Круговая диаграмма расходов
- [ ] Список категорий с суммами
- [ ] Процент от общих расходов
- [ ] Можно выбрать период
