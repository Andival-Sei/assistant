# Модуль: Assistant

> **Статус:** In Progress  
> **Приоритет:** Medium  
> **Зависимости:** Auth (Supabase)

## Обзор

Модуль помощника позволяет пользователям создавать задачи и напоминания с AI-разбором естественного языка, голосовым вводом и push-уведомлениями.

## Цели

- Создание и управление списками покупок
- Напоминания о событиях и задачах
- Push-уведомления
- Шаринг списков с семьёй

## Scope (планируется)

### В scope

- Списки покупок с чекбоксами
- Напоминания с датой и временем
- Push-уведомления (PWA)
- Шаринг списков в рамках семьи

### Вне scope

- Интеграция с календарями (Google, Apple)
- Голосовой ввод и преобразование в задачу
- Автоматическое добавление товаров

## Планируемые функции

### Списки

- Создание списка с названием
- Добавление элементов (товары, задачи)
- Отметка выполненных
- Шаринг с членами семьи
- Категоризация списков

### Напоминания

- Создание напоминания из произвольного текста
- AI-разбор даты/времени и формирование черновика
- Ручное подтверждение и редактирование перед сохранением
- Повторяющиеся напоминания (RRULE)
- Push-уведомления

## Технические заметки

- PWA для push-уведомлений
- Service Worker для фоновой синхронизации
- Supabase Edge Functions для отправки уведомлений (или отдельный backend)

## Интеграция с другими модулями

- **Family** — списки и напоминания можно шарить с семьёй
- **Finance** — напоминания о регулярных платежах (будущее)

## Статус

Реализован MVP умных задач (text/voice capture, AI parse draft, recurring reminders, push dispatch). Дальше — улучшение качества распознавания и расширение сценариев семейного шаринга.
