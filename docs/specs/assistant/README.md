# Модуль: Assistant

> **Статус:** Stub  
> **Приоритет:** Medium  
> **Зависимости:** Auth (Supabase)

## Обзор

Модуль помощника позволяет пользователям создавать списки покупок, напоминания и получать уведомления. В текущей версии — заглушка для будущей реализации.

## Цели

- Создание и управление списками покупок
- Напоминания о событиях и задачах
- Push-уведомления
- Шаринг списков с семьёй

## Scope (планируется)

### В scope

- Списки покупок с чекбоксами
- Напоминания с датой и временем
- Push-уведомления (PWA)
- Шаринг списков в рамках семьи

### Вне scope

- Интеграция с календарями (Google, Apple)
- Голосовой ввод
- Автоматическое добавление товаров

## Планируемые функции

### Списки

- Создание списка с названием
- Добавление элементов (товары, задачи)
- Отметка выполненных
- Шаринг с членами семьи
- Категоризация списков

### Напоминания

- Создание напоминания с текстом
- Установка даты и времени
- Повторяющиеся напоминания
- Push-уведомления

## Технические заметки

- PWA для push-уведомлений
- Service Worker для фоновой синхронизации
- Supabase Edge Functions для отправки уведомлений (или отдельный backend)

## Интеграция с другими модулями

- **Family** — списки и напоминания можно шарить с семьёй
- **Finance** — напоминания о регулярных платежах (будущее)

## Статус

Модуль находится в стадии планирования. Реализация запланирована на Фазу 2 (см. [roadmap](../../meta/roadmap.md)).
